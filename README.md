# Student Result Calculator

A simple web-based mini project to calculate student total marks, percentage, and result.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Enter marks for multiple subjects
- Calculates total marks
- Calculates percentage
- Displays pass or fail result

## How to Run
1. Download or clone the repository
2. Open `index.html` in a web browser
3. Enter marks and click Calculate

## Output
The application displays total marks, percentage, and result instantly.
